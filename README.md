# ✝ Daily Bread - 오늘의 성경 말씀

매일 새로운 성경 말씀으로 하루를 시작하세요.

## 🚀 바로가기
**[웹사이트 접속하기](https://ramiteacher.github.io/bible/)**

## ✨ 주요 기능
- **100개 이상의 성경 구절**: 창세기부터 요한계시록까지 다양한 구절이 랜덤하게 표시됩니다.
- **다크/라이트 모드**: 헤더 우측의 달/해 아이콘을 클릭하여 테마를 바꿀 수 있습니다.
- **말씀 저장**: 마음에 드는 구절을 별도로 저장하고 관리할 수 있습니다.
- **공유하기**: 친구와 오늘의 말씀을 공유해보세요.
- **반응형 디자인**: 모바일과 PC 환경 모두에서 최적화된 고급스러운 디자인을 제공합니다.

## 🛠 사용 방법
1. 웹사이트에 접속하면 자동으로 랜덤 성경 말씀이 표시됩니다.
2. "새로운 말씀 보기" 버튼을 클릭하여 다른 구절을 확인할 수 있습니다.
3. "이 구절 저장하기" 버튼을 클릭하여 마음에 드는 구절을 저장할 수 있습니다.
